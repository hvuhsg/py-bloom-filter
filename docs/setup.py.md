

# easy-bloom-filter

easy-bloom-filter is a python implementation of bloom filter.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install easy-bloom-filter.

```bash
pip install easy-bloom-filter
```

## Usage

```python
import easy-bloom-filter

# code away!
```
